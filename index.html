KMU Landingpage – Lead-Funnel

Scope: Frontend, SEO-Basics, Performance-Optimierung

Frontend
Load (LCP) 3.8s → 1.6s
Conversion 1.2% → 2.3%
Bounce 62% → 44%

Problem

Langsame Seite, unklarer Call-to-Action, Traffic ohne Leads.

Entscheidung

UI verschlankt, Above-the-fold optimiert, Form-Flow reduziert, Performance-Budget eingeführt.

Umsetzung

  • Component-first Layout, mobile-first
  • Bildoptimierung + Lazy Loading
  • SEO-Grundstruktur + Meta/Schema light
Proof: Analytics / Lighthouse / Search Console (auf Anfrage).

Corporate Site – Betrieb & Stabilität

Scope: Frontend + Hosting, Deployment, Monitoring

Hosting
Uptime 97.9% → 99.95%
Deploys/Monat 1 → 8
Incidents 6 → 1

Problem

Manuelle Deploys, Ausfälle, keine klare Ownership im Betrieb.

Entscheidung

Deployment-Pipeline + Monitoring + saubere Rollback-Strategie.

Umsetzung

  • Automatisierte Deployments
  • Uptime-Checks + Alerting
  • SSL/Domain/DNS sauber dokumentiert
Proof: Status-Monitoring/Logs (auf Anfrage).

Web-App – Admin + API

Scope: Full-Stack, Auth, CRUD-Flows, Admin UI

Full-Stack
Bearbeitungszeit 18m → 6m
Fehlerquote 9% → 2%
Time-to-Ship 6w → 3w
